# 機能一覧書 (機能一覧書（最新） - Book1 (1).pdf)

## F01 ユーザー認証機能
- **F01-1 ユーザー登録機能**
  - `F01-1-1 ユーザー情報入力処理`: メールアドレスとパスワードを入力、バリデーション。
  - `F01-1-2 MySQLアカウント作成処理`: MySQLでアカウントを新規作成。
  - `F01-1-3 パスワードリセット処理`: メールでリセットリンクを送付・仮パスワード発行。
- **F01-2 ユーザー認証処理**
  - `F01-2-1 ログイン処理`: MySQLによる認証。
  - `F01-2-2 ログアウト処理`: セッションを破棄しログイン画面へ。
- **F01-3 ユーザー退会処理**
  - `F01-3-1 Firebaseファイル削除`: Storage上のユーザーファイルを削除。
  - `F01-3-2 MySQLからユーザー削除`: MySQL上の関連情報をCascade Deleteで削除。

## F02 管理者ユーザー機能（Admin）
- **F02-1 ユーザー削除処理**
  - `F02-1-1 ユーザー選択処理`: 管理画面で削除対象ユーザーを選択。
  - `F02-1-2 Firebaseファイル削除`: Storage上のユーザーファイルを削除。
  - `F02-1-3 MySQLからユーザー削除`: DBからユーザーをCascade Deleteで削除。
- **F02-2 お問い合わせ表示機能**
  - `F02-2-1 お問い合わせ表示`: 受信したお問い合わせをリスト表示。
  - `F02-2-2 ページ分割機能`: ページネーションの設定。
  - `F02-2-3 お問い合わせ削除機能`: 不要な問い合わせを削除。

## F03 PDFアップロード＋AI注釈
- **F03-1 PDFアップロード機能**
  - `F03-1-1 PDFファイルアップロード`: PDFをアップロードしページ単位で画像(png)化。
  - `F03-1-2 Firebaseへの保存とURL取得`: Storageに保存しHTML用・管理用URL取得。
  - `F03-1-3 メタ情報のDB登録`: 画像URL、ページ番号をMySQL保存。
  - `F03-1-4 OCRによる文字情報抽出`: 画像をOCR処理しテキスト抽出。
- **F03-2 AI注釈生成機能**
  - `F03-2-1 テキスト送信処理`: OCRテキストをAIのAPIにPOSTし、JSONで受け取る。
- **F03-3 JSON パース機能**
  - `F03-3-1 JSON 構造の解析`: AIからのJSON結果を解析・抽出。
  - `F03-3-2 テキスト整形と保存`: タイトル・解説文などを整形しDB保存。

## F04 階層カテゴリ管理機能
- **F04-1 カテゴリ管理**
  - `F04-1-1 カテゴリ追加`: フォルダ構造2階層まで追加。
  - `F04-1-2 カテゴリ編集`: 既存カテゴリ名変更。
  - `F04-1-3 カテゴリ削除`: ID指定で削除。

## F05 ノート閲覧・編集機能
- **F05-1 ノート閲覧機能**
  - `F05-1-1 ログインＩＤ情報取得`: 必要なデータを初期取得。
  - `F05-1-2 ノート本文・画像の取得`: DB/Firebaseからデータを取得表示。キャッシュ利用。
  - `F05-1-3 キャッシュ管理`: Cache Storage + sessionStorage利用。
- **F05-2 ノート編集機能**
  - `F05-2-1 ノート編集フォームの表示`: タイトル・本文を編集。
  - `F05-2-2 ノート内容の保存`: 編集内容をDBにUPDATE。
- **F05-3 ファイル・ページ削除機能**
  - `F05-3-1 ファイル・ページ削除`: ページ単位での画像削除・DB情報削除。

## F06 プロンプト追加機能
- **F06-1 AI再注釈機能**
  - `F06-1-1 プロンプト入力`: AIへの追加指示テキストを入力。
  - `F06-1-2 プロンプトの保存`: デフォルト設定としてDB保存。

## F07 アクセス権管理機能・F08 問い合わせ機能
- **F07-1 アクセス権チェック**
  - `F07-1-1 アクセス権の確認処理`: 所有者のみ編集・削除可。
  - `F07-1-2 アクセス制御の適用`: 未ログイン時はTOPリダイレクト。
- **F08-1 お問い合わせ機能**
  - `F08-1-1 ログイン判別機能`: ログイン時は送信者を自動入力。
  - `F08-1-2 お問い合わせ送信機能`: 問い合わせをDB保存。
