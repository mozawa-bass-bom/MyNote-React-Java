# 企画書

## アプリ名
【授業ノートアプリ】
作成者: zd1U05 小澤 基良

## 概要
授業を受ける時にノートを取る際に、生成AIにPDFを読み込ませて授業内容を追加でメモをしていくという方法でやっていたが、難しい所になると詳細な解説が欲しい時が多々あった。
また、下準備に時間を取られることも煩わしかったため、PDFを1枚読み込ませたら自動でページ分割し、1ページずつ生成AIに送って返事として解説テキストを出力し、授業前の準備を1回で済ませるシステムを作成する。

- 事前にプロンプトを設定しておくが、独自に設定も可能にする可能性がある。
- 派生機能として、目次とページ数の作成（「フレームワーク実習/MyBatis入門」「Java実習/分岐処理２」等の階層構造作成）が考えられる。
- ログイン機能やエクスポート機能の実装も検討する。

## 使用技術（仮）
- Java (SpringBoot)
- ThymeLeaf
- bootStrap (sassでのカスタム)
- Apache PDFBox (PDF分割)
- MyBatis
- MySQL (ログイン認証用。将来的な他アプリとの統合の可能性に備える)
- FireBase (PDFデータ保存用)
- Google Cloud Vision API
- Google AI API (REST)

## システム概要
1. **ユーザーがPDFをアップロード**: フロントエンドからバックエンドのAPIへPDFファイルを送信。
2. **PDFのページ分割**: バックエンドのSpring BootアプリケーションがApache PDFBoxを使用してPDFをページごとに分割。
3. **AIへのリクエスト送信**: 分割された各ページのデータ（画像またはテキスト）をGoogle AI APIに送信。
4. **AIからの解説受信**: Google AI APIから解説テキストを受信。
5. **データ保存**: 生成された解説テキストをデータベースに保存し、ユーザーが参照・編集できるようにする。
6. **ユーザーへの表示**: フロントエンドで生成された解説テキストと元のPDFページを関連付けて表示。
