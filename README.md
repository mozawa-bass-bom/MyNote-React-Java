# MyNote

PDFを読み込み、OCRとAIを用いて解説付きノートを生成するアプリケーションです。  
Markdown形式で編集・保存でき、AIによる要約・解説生成を行うことができます。

# 工夫したポイント

## 状態管理
- ナビゲーション状態を **Jotai Atom** で管理
- コンポーネント間の状態共有をシンプルに設計
- 状態変更が即時UIに反映

## データ検証
- 新規登録時に **DBで重複チェック**
- データ整合性を確保
- 入力時にAPIを叩き存在確認。

## エディタ
- **MDXEditor** を採用
- Markdown形式の編集に対応
- 自動保存機能を実装

## AIのプロンプトとJsonモード
- AIにJsonモードで解説の作成を指示
- 生成したJsonからバックエンドでデータを整え供給

## 高速化
- fireBaseに画像、MySQLに文字データを格納
